# Better-Guitar-Tuner

## What's different?
As a guitar player who would **have a fever** when it comes to **music theory**, it's hard to handle all the **special tunings** other than the standard one. Instead of merely focusing on detecting the current pitch like any other guitar tuner, Better-Guitar-Tuner also provides modes to choose from common special tunings.

## Where to use it?
Currently, I am planning to launch a web page version first. If things works well,I would try to transport it to ios and android, as well as harmonyOS.
